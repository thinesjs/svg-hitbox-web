# SvgHitbox

Draw and manage hitboxes on SVG files. Export as JSON or TypeScript.

## Features

- Load any SVG and overlay rectangular or circular hitboxes
- Pan, zoom, select, resize, copy/paste, duplicate
- Custom fields per hitbox (key-value pairs)
- Right-click context menu with z-ordering, lock, flip
- Multi-select with marquee drag or Shift+click
- Import/export as `.json`, export as `.ts`

## Setup

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```

## Stack

React 18, TypeScript, Vite, Tailwind CSS v4, shadcn/ui
